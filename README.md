# About

A basic Pishock integration mod for Beat Saber. Currently only supports shock/vibrate on combo break.\
The mod uses the [PiShockLibrary](https://github.com/MxPuffin/PiShockLibrary) and some code from [REPOShock](https://github.com/MxPuffin/REPOShock) by [MxPuffin](https://github.com/MxPuffin). Icons provided by [icons8](https://icons8.com/)
# Configurations
Install the mod then add your credentials in to ShockSaber.json file in UserData. The rest is configured ingame.\
You can get your Pishock api and username key [here](https://login.pishock.com/account) and the access code [here](https://pishock.com/#/control)

## Ingame configurations
The duration/intensity and toggles can be adjusted with the ingame menu\
